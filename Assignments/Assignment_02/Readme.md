# Assignment 2

## Description

This folder contains **Assignment 2** for the **Introduction to AI Programming** course offered by the **Department of Electrical Engineering (Computing & AI), University of Engineering and Technology Peshawar, Jalozai Campus**.

The objective of this assignment is to strengthen Python programming skills by working with CSV files, lists, and dictionaries without using external libraries such as **pandas**. The assignment demonstrates how to manually read and organize data using Python's built-in `csv` module.

The program stores each column of the dataset in separate lists and performs manual grouping of electricity consumption data based on both **hour** and **day**. It then calculates the average and peak consumption for each hour, as well as the total and average consumption for each day. The results are displayed in a clean, organized, and easy-to-read format.

This assignment provides practical experience in data organization, dictionary manipulation, statistical calculations, and structured Python programming, which are essential skills for data analysis and AI applications.

- `data.csv` – Input dataset
