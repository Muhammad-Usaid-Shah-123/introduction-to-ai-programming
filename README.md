# 📖 About

Artificial Intelligence (AI) is one of the most rapidly evolving fields in computer science and engineering. It enables machines to perform tasks that typically require human intelligence, such as reasoning, learning, decision-making, and problem-solving. This course provides a solid foundation in AI programming by combining theoretical concepts with practical Python programming.

Throughout this course, I explore the core principles of Artificial Intelligence while developing programming skills that are essential for building intelligent applications. The repository reflects my progress, learning experience, and continuous improvement as I complete laboratory exercises, assignments, and engineering projects.
