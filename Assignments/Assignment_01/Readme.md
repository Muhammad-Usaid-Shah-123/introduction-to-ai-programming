# Assignment 1

## Overview

This folder contains **Assignment 1** for the **Introduction to AI Programming** course offered by the **Department of Electrical Engineering (Computing & AI), University of Engineering and Technology Peshawar, Jalozai Campus**.

The purpose of this assignment is to introduce students to Python file handling and basic data analysis using the built-in `csv` module. The assignment emphasizes manual data processing without relying on external libraries such as **pandas**.

## Assignment Tasks

The following tasks are completed in this assignment:

- Load the CSV file manually using the `csv` module.
- Store the **consumption** column in a list of floating-point values.
- Calculate the total consumption.
- Calculate the average consumption.
- Determine the maximum and minimum consumption values.
- Convert hourly consumption values into **kWh** using a list comprehension.
- Display all results in a clean and organized format.

## Learning Objectives

After completing this assignment, students will be able to:

- Read and process CSV files using Python.
- Store and manipulate numerical data using lists.
- Perform basic statistical calculations.
- Apply list comprehensions for data transformation.
- Write clean, readable, and well-structured Python programs.

## Technologies Used

- Python 3
- CSV Module (Built-in Python Library)
