# Assignment 3

## Description

This folder contains **Assignment 3** for the **Introduction to AI Programming** course offered by the **Department of Electrical Engineering (Computing & AI), University of Engineering and Technology Peshawar, Jalozai Campus**.

The objective of this assignment is to develop Python programming skills by working with multiple CSV files and user-defined functions. The program manually loads load and electricity price data, stores them in separate lists, and calculates the hourly and total electricity bill. This assignment demonstrates the practical use of file handling, lists, functions, and basic mathematical operations in Python.

## Files

- `Assignment_3.py` – Python source code
- `load.csv` – Load data
- `price.csv` – Electricity price data
