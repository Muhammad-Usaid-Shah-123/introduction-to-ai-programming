# Introduction to AI Programming

Welcome to the **Introduction to AI Programming** repository. This repository contains all of my coursework, laboratory work, assignments, open-ended laboratory activities, reports, and engineering projects completed during the **Introduction to AI Programming** course at the **University of Engineering and Technology Peshawar, Jalozai Campus**. It demonstrates my learning journey in Python programming, Artificial Intelligence fundamentals, and practical problem-solving through structured coursework and hands-on implementation.

# 📖 About

Artificial Intelligence (AI) is one of the fastest-growing fields in computer science and engineering. It focuses on developing intelligent systems capable of learning, reasoning, decision-making, and solving complex problems. Python is one of the most popular programming languages for AI because of its simplicity, readability, and extensive libraries.

This repository documents my academic progress throughout the semester by organizing laboratory reports, programming exercises, assignments, open-ended labs, and engineering projects in a professional manner. It serves as both a learning portfolio and a reference for future AI and Python development.
