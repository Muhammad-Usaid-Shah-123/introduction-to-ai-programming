# Open Ended Lab (OEL)

## Description

This folder contains the **Open Ended Lab (OEL)** for the **Introduction to AI Programming** course offered by the **Department of Electrical Engineering (Computing & AI), University of Engineering and Technology Peshawar, Jalozai Campus**.

The objective of this lab is to analyze the electricity consumption patterns of **Building 2** using the **CityLearn 2022** dataset. The project demonstrates how Python can be used to process real-world energy data, perform statistical analysis, and identify consumption trends without relying on advanced data analysis libraries.

The program manually loads the dataset using Python file handling techniques and stores electricity consumption values in a list. It then performs annual, monthly, daily, and seasonal energy consumption analysis. In addition, the program compares weekday and weekend electricity usage to identify behavioral patterns and operational trends.

The results provide valuable insights into building energy consumption, helping users understand electricity demand over different time periods. This lab strengthens Python programming skills while introducing practical concepts of energy data analysis, file handling, loops, lists, conditional statements, and basic statistical computations.

## Features

- Load the CityLearn 2022 Building 2 dataset.
- Read data using basic Python file handling.
- Calculate annual electricity consumption.
- Analyze monthly electricity usage.
- Compute daily energy consumption.
- Compare seasonal energy consumption.
- Analyze weekday and weekend electricity usage.
- Display results in a clean and organized format.
- Interpret energy consumption trends.
- Draw conclusions based on the analysis.
